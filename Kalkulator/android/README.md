# 📱 Android приложение

## 🏗️ Структура проекта

```
📦 android/
├── 📱 app/                        # Основное приложение
│   ├── 🎨 ui/                     # Пользовательский интерфейс
│   │   ├── 📱 screens/           # Экраны приложения
│   │   ├── 🧩 components/        # Переиспользуемые компоненты
│   │   ├── 🎨 themes/            # Темы и стили
│   │   └── 🎭 animations/        # Анимации
│   ├── 🧠 data/                  # Слой данных
│   │   ├── 📊 models/            # Модели данных
│   │   ├── 🗄️ database/          # Локальная база данных
│   │   ├── 🌐 api/               # API клиент
│   │   └── 🔄 repositories/       # Репозитории
│   ├── 🏗️ domain/                # Бизнес-логика
│   │   ├── 🧮 entities/          # Сущности
│   │   ├── 🔧 usecases/          # Случаи использования
│   │   └── 🔄 services/           # Сервисы
│   ├── 🛠️ utils/                 # Утилиты
│   │   ├── 🔧 helpers/           # Вспомогательные функции
│   │   ├── 📏 units/             # Система единиц
│   │   ├── 🧮 formula/           # Движок формул
│   │   └── 🔐 security/          # Безопасность
│   └── 🧪 tests/                 # Тесты
├── 📚 libraries/                 # Библиотеки
│   ├── 🧮 formula-engine/        # Движок формул
│   ├── 📏 units-converter/       # Конвертер единиц
│   ├── 🎨 ui-components/         # UI компоненты
│   └── 🔐 auth/                  # Аутентификация
└── 📋 build.gradle               # Конфигурация сборки
```

## 🎯 Основные функции

### 📱 Экраны
- **Главный экран** - список вкладок
- **Экран вкладки** - формулы и расчеты
- **Экран формулы** - редактирование формулы
- **Экран настроек** - настройки приложения
- **Экран профиля** - управление аккаунтом

### 🧩 Компоненты
- **FormulaCard** - карточка формулы
- **InputField** - поле ввода
- **OutputField** - поле вывода
- **UnitSelector** - выбор единиц
- **CollapsibleSection** - сворачиваемая секция

### 🧮 Движок формул
- Парсинг математических выражений
- Поддержка функций и операторов
- Система единиц измерения
- Автоматическое вычисление зависимостей

## 🛠️ Технологии

- **Kotlin** - основной язык
- **Jetpack Compose** - UI фреймворк
- **Room** - локальная база данных
- **Retrofit** - HTTP клиент
- **Hilt** - внедрение зависимостей
- **Material Design 3** - дизайн система

## 🚀 Запуск

1. Откройте проект в Android Studio
2. Синхронизируйте Gradle
3. Запустите на устройстве или эмуляторе
