# 📚 Общие компоненты

## 🏗️ Структура проекта

```
📦 shared/
├── 📊 models/                     # Модели данных
│   ├── 👤 user/                  # Модель пользователя
│   ├── 📊 tab/                   # Модель вкладки
│   ├── 🧮 formula/              # Модель формулы
│   ├── 📏 value/                # Модель значения
│   └── 🔧 unit/                 # Модель единицы
├── 🧮 formula-engine/            # Движок формул
│   ├── 🔧 parser/               # Парсер формул
│   ├── 🧮 evaluator/            # Вычислитель
│   ├── 📏 units/                # Система единиц
│   └── 🔐 validator/            # Валидатор
├── 📏 units-converter/           # Конвертер единиц
│   ├── 📊 categories/            # Категории единиц
│   ├── 🔧 converters/           # Конвертеры
│   └── 🧪 tests/                # Тесты
├── 🎨 ui-components/             # UI компоненты
│   ├── 🧩 buttons/              # Кнопки
│   ├── 📱 inputs/               # Поля ввода
│   ├── 📊 displays/             # Отображение
│   └── 🎭 animations/          # Анимации
├── 🔐 auth/                     # Аутентификация
│   ├── 🔑 tokens/               # Токены
│   ├── 🔐 encryption/           # Шифрование
│   └── 🛡️ security/             # Безопасность
├── 🛠️ utils/                     # Утилиты
│   ├── 🔧 helpers/              # Вспомогательные функции
│   ├── 📏 math/                 # Математические функции
│   ├── 🔐 string/              # Строковые функции
│   └── 📊 validation/           # Валидация
├── 🧪 tests/                     # Тесты
│   ├── 🧪 unit/                # Модульные тесты
│   ├── 🧪 integration/         # Интеграционные тесты
│   └── 🧪 e2e/                 # End-to-end тесты
└── 📋 package.json              # Зависимости
```

## 🎯 Основные функции

### 📊 Модели данных
- Единые модели для всех платформ
- Типизация данных
- Валидация моделей
- Сериализация/десериализация

### 🧮 Движок формул
- Парсинг математических выражений
- Поддержка функций и операторов
- Система единиц измерения
- Автоматическое вычисление зависимостей

### 📏 Система единиц
- Категории единиц измерения
- Автоматическая конвертация
- Валидация единиц
- Поддержка пользовательских единиц

### 🎨 UI компоненты
- Переиспользуемые компоненты
- Адаптивный дизайн
- Темы и стили
- Анимации и переходы

### 🔐 Безопасность
- Шифрование данных
- Валидация входных данных
- Защита от инъекций
- Управление токенами

## 🛠️ Технологии

- **TypeScript** - типизация
- **Jest** - тестирование
- **ESLint** - линтинг
- **Prettier** - форматирование
- **Webpack** - сборка

## 🚀 Использование

### Установка
```bash
npm install
```

### Сборка
```bash
npm run build
```

### Тестирование
```bash
npm test
```

### Линтинг
```bash
npm run lint
```
