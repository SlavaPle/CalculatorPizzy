# 🌐 Веб-приложение

## 🏗️ Структура проекта

```
📦 web/
├── 📱 src/                        # Исходный код
│   ├── 🎨 components/             # React компоненты
│   │   ├── 🧩 ui/                 # UI компоненты
│   │   ├── 📱 pages/             # Страницы
│   │   ├── 🧮 formula/           # Компоненты формул
│   │   └── 🔧 layout/            # Компоненты макета
│   ├── 🧠 hooks/                  # React хуки
│   │   ├── 🔄 useApi/            # API хуки
│   │   ├── 🧮 useFormula/        # Хуки для формул
│   │   └── 🔐 useAuth/           # Хуки аутентификации
│   ├── 🏗️ services/              # Сервисы
│   │   ├── 🌐 api/               # API клиент
│   │   ├── 🧮 formula/           # Сервис формул
│   │   └── 🔐 auth/              # Сервис аутентификации
│   ├── 🧠 store/                  # Управление состоянием
│   │   ├── 📊 slices/            # Redux слайсы
│   │   ├── 🔄 middleware/        # Middleware
│   │   └── 🧪 selectors/         # Селекторы
│   ├── 🛠️ utils/                 # Утилиты
│   │   ├── 🔧 helpers/           # Вспомогательные функции
│   │   ├── 📏 units/             # Система единиц
│   │   ├── 🧮 formula/           # Движок формул
│   │   └── 🔐 security/          # Безопасность
│   ├── 🎨 styles/                # Стили
│   │   ├── 🎭 themes/            # Темы
│   │   ├── 🎨 components/        # Стили компонентов
│   │   └── 📱 responsive/        # Адаптивные стили
│   └── 🧪 tests/                 # Тесты
├── 📚 public/                     # Публичные файлы
│   ├── 🖼️ images/                # Изображения
│   ├── 🎨 icons/                 # Иконки
│   └── 📄 manifest.json          # Манифест
├── 📋 package.json               # Зависимости
├── 📋 tsconfig.json              # TypeScript конфиг
└── 📋 vite.config.ts             # Vite конфиг
```

## 🎯 Основные функции

### 📱 Страницы
- **Главная** - список калькуляторов
- **Калькулятор** - рабочее пространство
- **Формула** - редактор формулы
- **Профиль** - управление аккаунтом
- **Настройки** - настройки приложения

### 🧩 Компоненты
- **FormulaBuilder** - конструктор формул
- **UnitConverter** - конвертер единиц
- **FormulaPreview** - предпросмотр формулы
- **CalculationResult** - результат расчета
- **TemplateSelector** - выбор шаблонов

### 🧮 Упрощенный генератор формул
- Визуальный конструктор
- Готовые шаблоны
- Мастер создания
- Предпросмотр в реальном времени

## 🛠️ Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик
- **Material-UI** - компоненты
- **Redux Toolkit** - управление состоянием
- **Axios** - HTTP клиент
- **MathJax** - отображение формул

## 🚀 Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите в режиме разработки:
   ```bash
   npm run dev
   ```

3. Откройте http://localhost:3000
